HTML

HTML Stands for Hyper text markup language. This is a tag based language tags are created with <> signs.

For Example:

<p> is a paragraph</p>

There are special characters such as &lt; would be Less then sign

Inline vs Block

Inline means that is inlline with the text.

Block level forces anew line

Block Level: div, p, h1, h2, h3, h4, h5, h6, li

Inline elements are span, img, a, strong, em
